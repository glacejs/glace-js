forEachLanguage( [ctx] [, opts] [, fixtures], func)

Iterates test chunks through all languages specifed in config.

It's applicable for multilingual application. If list of languages is specified, it will be used firstly. Otherwise from configuration.

Parameters:
Name Type Argument Description
ctx object <optional>

Test case context.
opts object <optional>

Options.

Properties
Name Type Argument Default Description
video boolean <optional>
 false

Flag to involve video fixture. May be managed in config.
fixtures Array.<function()> <optional>

Involved fixtures list.
func function

Function with test steps.
Source:
  • globals.js, line 438
Example
test("Some test", ctx => {
    forEachLanguage(ctx, lang => {
        beforeChunk(() => {
            someFunc();
        });
        chunk("Chunk #1", () => {
            someFunc();
        });
        chunk("Chunk #2", () => {
            someFunc();

    });
});
